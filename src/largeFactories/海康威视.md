# 九、海康威视

## 1.knex 连接数据库响应回调

- Knex 是 SQL query builder，连接数据库后返回 Promise。
- 可通过 `then/catch` 或 `async/await` 处理查询结果和异常。

## 2.介绍异步方案

- 回调函数、Promise、Generator、async/await。
- 事件监听、定时器、网络请求等由宿主环境提供异步能力。

## 3.如何处理异常捕获

- 同步代码用 `try/catch`。
- Promise 用 `.catch()`，async/await 用 `try/catch`。
- 全局可监听 `window.onerror`、`unhandledrejection` 或 Node 的异常事件。

## 4.项目如何管理模块

- 按业务域或功能分层组织目录，如 components、services、store、utils。
- 使用 ES Module 导入导出，配合 alias 简化路径。
- 公共逻辑抽到独立模块，避免循环依赖。

## 5.前端性能优化

- 资源层面：压缩、缓存、CDN、按需加载。
- 渲染层面：减少重排重绘、虚拟列表、合成层动画。
- 代码层面：防抖节流、减少重复计算、避免内存泄漏。

## 6.JS 继承方案

- ES5 可用原型链继承、构造函数继承、组合继承、寄生组合继承。
- ES6 使用 `class extends`，底层仍依赖原型链。

## 7.如何判断一个变量是不是数组

- 推荐使用 `Array.isArray(value)`。
- 也可以用 `Object.prototype.toString.call(value) === '[object Array]'`。

## 8.变量 a 和 b，如何交换

- 可使用临时变量：`let t = a; a = b; b = t;`。
- ES6 可用解构：`[a, b] = [b, a]`。

## 9.事件委托

- 把子元素事件绑定到父元素上，通过事件冒泡统一处理。
- 优点是减少事件绑定数量，也能处理动态新增元素。

## 10.多个

- 如果指多个事件或多个节点，优先考虑批量处理和事件委托。
- 如果指多个异步任务，可用 `Promise.all` 或并发队列控制。

## 11.标签生成的 Dom 结构是一个类数组

- DOM 查询结果如 `NodeList`、`HTMLCollection` 是类数组。
- 它们有索引和 `length`，但不一定具备完整数组方法。

## 12.类数组和数组的区别

- 类数组有索引和长度，但原型上没有数组的全部方法。
- 数组可直接使用 `map`、`filter`、`reduce` 等方法。

## 13.dom 的类数组如何转成数组

- 使用 `Array.from(nodeList)`。
- 或使用扩展运算符 `[...nodeList]`。
- 也可用 `Array.prototype.slice.call(nodeList)`。

## 14.介绍单页面应用和多页面应用

- SPA 只有一个 HTML 入口，页面切换由前端路由完成。
- MPA 每个页面通常对应一个 HTML，切换页面会重新加载。
- SPA 体验更流畅，MPA 首屏和 SEO 通常更直接。

## 15.redux 状态树的管理

- 按业务模块拆分 reducer，再组合成根 reducer。
- 状态保持扁平化，避免深层嵌套。
- 派生数据通过 selector 计算，不必全部存入 state。

## 16.介绍 localstorage 的 API

- `localStorage.setItem(key, value)` 存储字符串。
- `localStorage.getItem(key)` 读取。
- `localStorage.removeItem(key)` 删除指定项。
- `localStorage.clear()` 清空全部数据。
